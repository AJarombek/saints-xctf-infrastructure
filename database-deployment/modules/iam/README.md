### Overview

Module for IAM roles and policies used by the database deployment AWS Lambda function.

### Files

| Filename                  | Description                                                                 |
|---------------------------|-----------------------------------------------------------------------------|
| `main.tf`                 | Main Terraform file that creates IAM roles and policies.                    |
| `policy.json`             | IAM policy JSON document.                                                   |
| `assume-role-policy.json` | IAM assume role policy JSON document.                                       |
