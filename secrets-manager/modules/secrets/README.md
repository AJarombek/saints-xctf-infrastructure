### Overview

Module for configuring AWS Secrets Manager.

### Files

| Filename            | Description                                                                                  |
|---------------------|----------------------------------------------------------------------------------------------|
| `main.tf`           | Main Terraform file that configures AWS Secrets Manager.                                     |
| `var.tf`            | Input variables for the AWS Secrets Manager module.                                          |
